# MangaShift — Project Brief

> Wizja produktu i kontekst landing page'a. Dokument produktowy, nie release-notes.

## Czym jest MangaShift

Automatyczny generator audiowizualnych adaptacji mangi. System przekształca statyczne
strony mangi/manhwy w dynamiczne wideo z narracją — pipeline AI: **detekcja → OCR →
tłumaczenie → synteza mowy (TTS) → automatyczny montaż wideo**.

## Problem

Manga jest w dużej mierze niedostępna dla osób z dysfunkcjami wzroku i ruchu. Ręczna
adaptacja do formy audiowizualnej jest droga i czasochłonna, więc praktycznie nie powstaje.

## Rozwiązanie i wartość

- **Automatyzacja** całego pipeline'u — radykalne skrócenie czasu adaptacji.
- **Obniżenie kosztów** względem ręcznej produkcji.
- **Dostępność** — nowa forma odbioru mangi dla osób z niepełnosprawnościami.
- Algorytmy dostosowane do specyfiki mangi (układ kadrów, kierunek czytania, dymki).

## Grupa docelowa

- **Główni beneficjenci:** osoby z dysfunkcjami wzroku i ruchu.
- **Kluczowi użytkownicy:** wydawnictwa mangi, NGO, instytucje publiczne, twórcy.

## Landing page — cel i struktura

Cel obecnej strony: **wizytówka startupu + budowanie listy mailowej na beta-testy.**
One-page scroll, dark/light, bogate efekty wizualne (liquid glass, animacje).

Sekcje: **Hero → Demo → About → Beta/Contact → FAQ → Footer.**

- **Hero** — wartość produktu + CTA do zapisu na beta.
- **Demo** — pokazanie działania (preview, wideo).
- **About** — problem, rozwiązanie, dla kogo.
- **Beta/Contact** — zapis na listę beta-testerów (email) + kanały kontaktu.
- **FAQ** — najczęstsze pytania.

## Roadmap kamieni milowych (produkt)

1. **KM1** — moduł ekstrakcji i strukturyzacji treści (detekcja + OCR).
2. **KM2** — moduł tłumaczenia i generowania narracji (LLM + TTS).
3. **KM3** — pipeline automatycznego montażu wideo.
4. **KM4** — finalizacja.

## Plany rozwoju (po obecnej fazie)

- Własna domena + przeniesienie z GitHub Pages na docelowy hosting.
- Import istniejącej listy maili (~30) do serwisu formularzy.
- Redesign sekcji w kierunku stylu suminagashi (wg dostarczonych makiet).
